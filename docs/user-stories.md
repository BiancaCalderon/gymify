# Historias de Usuario - Gymify

## HU-01 Registro de usuario

Como usuario nuevo,

Quiero crear una cuenta en Gymify,

Para registrar mis entrenamientos y dar seguimiento a mi progreso.

### Criterios de aceptación

- El usuario puede registrarse mediante correo y contraseña.
- El correo debe ser único.
- El sistema confirma el registro exitoso.

---

## HU-02 Inicio de sesión

Como usuario registrado,

Quiero iniciar sesión en la aplicación,

Para acceder a mis datos y progreso.

### Criterios de aceptación

- El usuario ingresa correo y contraseña.
- El sistema valida las credenciales.
- Se concede acceso al perfil.

---

## HU-03 Registrar entrenamiento

Como usuario,

Quiero registrar una sesión de entrenamiento,

Para llevar control de mi actividad física.

### Criterios de aceptación

- Se registra fecha del entrenamiento.
- Se registra duración.
- Se registra tipo de entrenamiento.
- El registro queda almacenado.

---

## HU-04 Mantener una racha

Como usuario,

Quiero que mis entrenamientos generen una racha diaria,

Para motivarme a mantener la constancia.

### Criterios de aceptación

- La racha aumenta al registrar actividad válida.
- La racha se reinicia al incumplir las reglas establecidas.
- La racha actual es visible en el perfil.

---

## HU-05 Obtener experiencia (XP)

Como usuario,

Quiero recibir puntos de experiencia por entrenar,

Para visualizar mi progreso.

### Criterios de aceptación

- Cada actividad otorga XP.
- El XP acumulado se almacena.
- El usuario puede visualizar su XP total.

---

## HU-06 Subir de nivel

Como usuario,

Quiero subir de nivel conforme acumulo XP,

Para sentir progreso dentro de la aplicación.

### Criterios de aceptación

- Existen umbrales definidos para cada nivel.
- El sistema actualiza automáticamente el nivel.

---

## HU-07 Desbloquear logros

Como usuario,

Quiero obtener logros al cumplir ciertos objetivos,

Para recibir reconocimiento por mi esfuerzo.

### Criterios de aceptación

- Los logros poseen condiciones específicas.
- El sistema detecta automáticamente el cumplimiento.

---

## HU-08 Visualizar perfil

Como usuario,

Quiero consultar mi perfil,

Para observar mis estadísticas y progreso.

### Criterios de aceptación

- Se muestra nivel.
- Se muestra XP.
- Se muestra racha actual.
- Se muestran logros obtenidos.

---

## HU-09 Compartir logros

Como usuario,

Quiero publicar mis logros en el feed social,

Para compartir mi progreso con otros usuarios.

### Criterios de aceptación

- El usuario puede crear publicaciones.
- Se muestran en el feed.

---

## HU-10 Interactuar con publicaciones

Como usuario,

Quiero reaccionar y comentar publicaciones,

Para interactuar con la comunidad.

### Criterios de aceptación

- Se permiten likes.
- Se permiten comentarios.

---

## HU-11 Participar en retos grupales

Como usuario,

Quiero unirme a retos grupales,

Para aumentar mi motivación.

### Criterios de aceptación

- El usuario puede unirse a un reto.
- El progreso es visible.

---

## HU-12 Consultar rankings

Como usuario,

Quiero visualizar rankings,

Para comparar mi progreso con otros participantes.

### Criterios de aceptación

- Los rankings muestran posición.
- Se actualizan automáticamente.
