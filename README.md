# Gymify

Sistema de gamificación social para incrementar la adherencia al entrenamiento en usuarios de gimnasios en Guatemala.

Proyecto desarrollado para el curso de Diseño e Innovación, Universidad del Valle de Guatemala.

## Estructura del repositorio

```
gymify/
├── backend/      Backend en Golang (API REST)
├── frontend/     App móvil híbrida (React Native + Expo)
└── docs/         Documentación del proyecto (arquitectura, ERD, etc.)
```

## Stack tecnológico

- **Frontend:** React Native (Expo)
- **Backend:** Go (Golang)
- **Base de datos:** PostgreSQL
- **Autenticación:** JWT + OAuth 2.0 (Google)

## Metodología

El desarrollo sigue Scrum, organizado en 6 sprints de 3 semanas (18 semanas totales). Ver [docs/sprints.md](docs/sprints.md) para el detalle de cada sprint.
